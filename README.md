# 图片标注工具 · 增强版

> 一个基于 Canvas 的轻量级图片标注工具，支持矩形、圆形、旋转矩形三种标注，并提供全局遮罩（自动边界裁剪）、独立标注颜色、批量导出（PNG/JPEG）等功能。同时，它也可以作为**批量图片格式转换器**使用。

---

## 📌 项目简介

本工具是一个纯前端 HTML/CSS/JavaScript 应用，无需后端服务，所有数据保存在内存中。您可以通过“选择文件夹”加载多张图片，在画布上绘制标注形状，利用遮罩统一裁剪区域，并导出为 PNG 或 JPEG（带质量压缩）。**当不启用遮罩时，工具可一键将文件夹内所有图片批量转换为指定格式。**

## ✨ 主要功能

| 功能模块         | 说明                                                         |
| ---------------- | ------------------------------------------------------------ |
| **图片管理**     | 选择文件夹，自动加载所有支持的图片（jpg/png/webp/bmp/tiff等），左侧缩略图列表，点击切换 |
| **标注工具**     | 三种形状：矩形（Rect）、圆形（Circle）、旋转矩形（Rotated）  |
| **独立标注颜色** | 每个标注可单独设置颜色（选中后通过颜色选择器修改），未选中时修改的是新建标注的默认颜色 |
| **线型切换**     | 实线 / 虚线，全局生效                                        |
| **标注编辑**     | 单击选中标注，拖拽控制点调整大小（或旋转矩形角度），按 `Delete` 键删除 |
| **全局遮罩**     | 尺寸（宽高）全局统一，位置（左上角）每张图片独立；支持归一化（百分比），自动适配不同尺寸图片；**非归一化模式下自动裁剪超出边界的部分**，确保绘制和导出均无无效区域 |
| **遮罩开关**     | 可随时启用/禁用遮罩，禁用后不影响已有数据                    |
| **撤销操作**     | 支持撤销标注的添加、删除、编辑以及遮罩调整（Ctrl+Z）         |
| **导出当前图片** | 弹出对话框，可选择格式（PNG/JPEG）、JPEG质量、是否裁剪（遮罩）、是否绘制标注 |
| **批量导出**     | 一键导出所有图片，统一应用上述导出选项，打包为 ZIP 文件。**当关闭遮罩时，即成为批量格式转换工具** |
| **键盘快捷键**   | `Delete` 删除标注，`Ctrl+Z` 撤销，`Ctrl+S` 导出当前图片（已移除数字键切换工具，避免输入框冲突） |

## 🛠️ 技术栈

- **HTML5** + **CSS3**（Flexbox/Grid 布局，暗色主题）
- **原生 JavaScript**（ES6+，无第三方框架）
- **Canvas 2D API**（绘制与交互）
- **JSZip**（CDN 加载，用于批量导出 ZIP）

## 📁 文件结构
### 各文件职责

- **index.html**  
  定义工具栏、左侧图片列表、画布、状态栏以及导出模态框的结构。外部资源（JSZip）通过 CDN 加载。

- **style.css**  
  包含全局样式、组件样式（按钮、遮罩参数、模态框）、以及适配移动端的响应式规则。

- **script.js**  
  核心逻辑文件，包含：
  - `state` 全局状态对象
  - 图片加载、缩略图渲染
  - 标注绘制与交互（鼠标事件）
  - 遮罩绘制与交互（鼠标事件，含边界裁剪）
  - 标注样式控制（独立颜色、线型）
  - 撤销功能（标注及遮罩调整）
  - 导出函数（单张/批量，支持格式、质量、裁剪、标注开关）
  - 模态框控制
  - 快捷键绑定（已移除数字键）
  - 窗口自适应

## 🚀 快速上手

1. 将三个文件放在同一目录下。
2. 使用现代浏览器（Chrome / Edge / Firefox）打开 `index.html`。
3. **加载图片**：点击“📁 文件夹”，选择包含图片的目录（支持子目录，但仅加载根目录下的图片文件）。
4. **浏览图片**：左侧列表显示所有图片，点击缩略图切换。
5. **绘制标注**：选择工具（矩形/圆形/旋转矩形），在图片上拖拽绘制。
6. **编辑标注**：点击已绘制的形状可选中，拖拽黄色控制点调整大小/角度；选中后通过颜色选择器修改该标注颜色。
7. **设置遮罩**：点击“🎭 遮罩”进入遮罩模式，拖拽绘制遮罩，或调整现有遮罩；遮罩尺寸全局统一，位置每图独立；可启用/禁用或切换归一化。
8. **导出单张**：点击“💾 导出当前”，选择格式、质量、是否裁剪和绘制标注。
9. **批量导出/格式转换**：点击“📦 全部打包”，设置统一参数，系统将生成 ZIP 压缩包。**关闭遮罩且不绘制标注时，即实现批量格式转换。**

## ⌨️ 快捷键

| 按键 | 功能 |
|------|------|
| `Delete` / `Backspace` | 删除当前选中的标注 |
| `Ctrl+Z` | 撤销上一步操作（标注或遮罩调整） |
| `Ctrl+S` | 打开导出对话框（当前图片） |

> ⚠️ 注意：数字键 1、2、3 不再用于切换工具，以避免在遮罩输入框中输入数字时误触。

## 🧠 核心设计

- **状态管理**：所有可变状态集中在 `state` 对象中，`drawScene()` 统一触发重绘，避免 DOM 直接操作。
- **坐标转换**：鼠标事件均通过 `canvasToImage()` 和 `imageToCanvas()` 转换，保证标注坐标基于原始图片尺寸，支持图片缩放居中。
- **遮罩边界裁剪**：提供 `clampMaskRect` 函数，确保在任何模式下（特别是非归一化像素模式）遮罩都不会超出图片边界，绘制和导出均只保留有效区域。
- **独立颜色存储**：每个标注对象带有 `color` 属性，选中后修改颜色仅影响该标注；未选中时修改的是 `defaultAnnotationColor`，用于新建标注。
- **输入框焦点保护**：遮罩输入框获得焦点时，`updateMaskInputs()` 不会刷新值，防止打断用户输入。

---

## 📝 注意事项

- **导出 JPEG 透明背景**：如果原图是带透明通道的 PNG，导出为 JPEG 时透明区域会填充为黑色（JPEG 不支持透明度）。如需保留透明背景，请选择 PNG 格式。
- **遮罩与格式转换**：若将本工具用作批量格式转换器，请确保遮罩处于禁用状态（取消勾选“启用”），并取消导出对话框中的“裁剪”和“绘制标注”选项。
- **文件命名**：导出图片会添加“标注_”前缀，批量导出时 ZIP 内文件名格式为 `标注_原文件名.png` 或 `.jpg`。
- **浏览器限制**：由于使用 `webkitdirectory` 属性，仅支持 Chrome、Edge、Firefox 等现代浏览器，不支持 Safari 旧版本。
